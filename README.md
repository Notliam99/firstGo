### To Run The Project

**Requirements**

- go *(golang)*

**Run The Program**
```bash 
# To Just Run 
Go ./main.go

# To Build Run
go build ./main.go
# You Will Find The Compiled Binary in the source Folder

# Run The Binary On Linux
./main

# Run On Windows Simply navigate to it and double click
./main.exe
```
